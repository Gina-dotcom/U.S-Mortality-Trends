# U.S-Mortality-Trends
A data analysis project exploring how the leading causes of death in the United States changed between 2020 and 2023. The repository includes scripts for data extraction, cleaning, trend analysis, and visualizations that highlight shifts in mortality patterns during the COVID‑19 era.
# Project Objectives
To examine the overall mortality trends
Measure the month-to-month trends (monthly patterns)
To examine the relationships between causes of death
To analyse the percentage contribution of each cause of death
To examine the outliers and anomalies
# Data source
Data.CDC.gov
# Visual Insights
Natural cause attributes to the highest cause of death in the united states accounting approximately 11 million deaths. This is followed by Heart Diseases contributing approximately 3 million deaths, while deaths from Homicides accounts the lowest about 9 thousand.
Based on seasonal patterns; Januarary had the highest deaths about 2.5 million whereas Octomber recorded the lowest deaths of approximately 1.5 million.
